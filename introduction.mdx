## Overview

Welcome to Jayesh Chaudhari's documentation space. You use this platform to organize, manage, and share all your project documentation in one centralized location. It supports rich MDX content, making it easy to create interactive guides, API references, and tutorials with components like cards, steps, and tabs.

This space helps you streamline workflows, collaborate with teams, and keep everything version-controlled. Start by exploring key features below.

## Get Started in Minutes

Follow these steps to create your first documentation page.

<Steps>
  <Step title="Create a New Page" icon="plus">
    Navigate to your project root and add a file like `my-guide.mdx`.
  </Step>
  <Step title="Add Frontmatter" icon="file-text">
    Start with YAML frontmatter:
    
````mdx
```yaml
---
title: My Guide
description: A sample page.
---
```
````
  </Step>
  <Step title="Write Content" icon="edit-3">
    Use H2 headings and components. Here's a multi-language example:
    
    <CodeGroup tabs="JavaScript,Python">
```javascript
console.log("Hello, Jayesh Chaudhari docs!");
```
```python
print("Hello, Jayesh Chaudhari docs!")
```
    </CodeGroup>
  </Step>
  <Step title="Preview and Deploy" icon="eye">
    Preview locally, then commit to deploy. Your docs go live at `https://docs.example.com`.
  </Step>
</Steps>
