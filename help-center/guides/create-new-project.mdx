## Quick Start Guide

<Steps>
  <Step title="Log in to your account">
    Go to `https://docs.jayeshchaudhari.com` and sign in.
  </Step>
  <Step title="Click 'New Project'">
    Find the `New Project` button on the dashboard and click it.
  </Step>
  <Step title="Fill in project details">
    Enter a project name and optional description.
  </Step>
  <Step title="Start documenting">
    Add pages and content right away using the editor.
  </Step>
</Steps>
